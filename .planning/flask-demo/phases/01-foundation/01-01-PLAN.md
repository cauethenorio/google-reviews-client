**flask-example/app.py** (per D-07, D-08, D-10, D-11):
- `REQUIRED_ENV_VARS = ["GOOGLE_CLIENT_ID", "GOOGLE_CLIENT_SECRET", "SECRET_KEY"]`
- `_check_required_env_vars()` iterates over list, raises `RuntimeError(f"Required environment variable {var} is not set. See .env.example.")` if `os.environ.get(var)` is falsy
- `_make_fernet_key(secret: str) -> bytes` does `base64.urlsafe_b64encode(hashlib.sha256(secret.encode()).digest())`
- `create_app()`:
  1. Calls `_check_required_env_vars()`
  2. Creates `app = Flask(__name__)`
  3. Sets `app.secret_key = os.environ["SECRET_KEY"]`
  4. Sets `app.config["FERNET"] = Fernet(_make_fernet_key(os.environ["SECRET_KEY"]))`
  5. Sets `app.config["GOOGLE_CLIENT_ID"] = os.environ["GOOGLE_CLIENT_ID"]`
  6. Sets `app.config["GOOGLE_CLIENT_SECRET"] = os.environ["GOOGLE_CLIENT_SECRET"]`
  7. Imports and registers `auth_bp` from `auth` and `views_bp` from `views`
  8. Returns `app`
- Imports: `os`, `base64`, `hashlib`, `Fernet` from `cryptography.fernet`, `Flask` from `flask`
- Flask auto-loads `.env` via python-dotenv when using `flask run` -- no manual `load_dotenv()` needed

**flask-example/cookies.py** (per D-05, D-06):
- `TOKEN_COOKIE_NAME = "greviews_tokens"`
- `COOKIE_MAX_AGE = 60 * 60 * 24 * 7` (7 days)
- `encrypt_tokens(token_dict: dict, fernet: Fernet) -> str`: JSON-encodes dict, encrypts with fernet, returns decoded string
- `decrypt_tokens(cookie_value: str, fernet: Fernet) -> dict | None`: try decrypt + JSON decode, return dict; except `(InvalidToken, json.JSONDecodeError, Exception)` return `None`
- Imports: `json`, `Fernet` and `InvalidToken` from `cryptography.fernet`
