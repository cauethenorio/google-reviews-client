**Helper function `_build_client_config(client_id, client_secret)`** returning:
```python
{
    "web": {
        "client_id": client_id,
        "client_secret": client_secret,
        "auth_uri": "https://accounts.google.com/o/oauth2/auth",
        "token_uri": "https://oauth2.googleapis.com/token",
    }
}
```

**OAUTHLIB_INSECURE_TRANSPORT:** At module level, set `os.environ["OAUTHLIB_INSECURE_TRANSPORT"] = "1"` when `os.environ.get("FLASK_DEBUG")` is truthy. This prevents InsecureTransportError during local HTTP development.

**`login_required` decorator** (per D-14):
- Read cookie via `request.cookies.get(TOKEN_COOKIE_NAME)`
- If no cookie: `redirect("/?error=session_expired")`
- Decrypt with `decrypt_tokens(cookie, current_app.config["FERNET"])`
- If None or `data.get("auth_status") != "authenticated"`: `redirect("/?error=session_expired")`
- Otherwise call the wrapped function

**`/login` route** (per D-09, D-10, D-11, D-12):
- Build client config from `current_app.config["GOOGLE_CLIENT_ID"]` and `GOOGLE_CLIENT_SECRET`
- Compute `redirect_uri` as `os.environ.get("REDIRECT_URI") or url_for("auth.callback", _external=True)`
- Create `Flow.from_client_config(client_config, scopes=SCOPES, redirect_uri=redirect_uri)`
- Call `flow.authorization_url(access_type="offline", prompt="consent")` which returns `(authorization_url, state)` and auto-generates PKCE code_verifier
- Build pending cookie data: `{"auth_status": "pending", "state": state, "verifier": flow.code_verifier}`
- Encrypt with `encrypt_tokens(pending_data, current_app.config["FERNET"])`
- Create redirect response to `authorization_url`
- Set cookie with: `httponly=True, secure=not current_app.debug, samesite="Lax", max_age=300` (5 min for pending state)
- Return response

**`/callback` route** (per D-12, D-13, D-15):
- Read and decrypt pending cookie. If missing: `redirect("/?error=missing_state")`. If invalid or not pending: `redirect("/?error=invalid_state")`
- Check `request.args.get("state") != pending["state"]`: `redirect("/?error=state_mismatch")`
- Check `"error" in request.args`: `redirect(f"/?error={request.args['error']}")`
- Reconstruct Flow with `state=pending["state"], code_verifier=pending["verifier"], autogenerate_code_verifier=False` -- CRITICAL: do NOT let it regenerate the verifier
- Call `flow.fetch_token(authorization_response=request.url)`
- Extract credentials: `flow.credentials`
- Build token data: `{"auth_status": "authenticated", "access_token": credentials.token, "refresh_token": credentials.refresh_token, "expiry": credentials.expiry.isoformat() if credentials.expiry else None}`
- Encrypt and set cookie with `max_age=COOKIE_MAX_AGE, httponly=True, secure=not current_app.debug, samesite="Lax"`
- Redirect to `/accounts` per D-15

**`/logout` route** (per D-17, D-18):
- Create redirect response to `/`
- Call `response.delete_cookie(TOKEN_COOKIE_NAME)`
- Return response

**Also update `.env.example`:** Add `# REDIRECT_URI=https://your-domain.com/callback` (commented out, optional) below the existing env vars, with a comment explaining it's for proxy environments per D-11.
